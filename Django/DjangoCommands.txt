Start project / create a django folder:   	      	django-admin startproject PROJECT_NAME 

For Astablish server conection: 	      	python manage.py runserver

Start program in App:     	                          	python manage.py startapp APP_NAME

In settings.py in installed apps add APP_NAME

Create url.py in App
